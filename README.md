# Angular CRUD - Sistema de Clientes

Projeto CRUD desenvolvido em Angular com Angular Material, focado em cadastro, listagem, edição e exclusão de clientes, utilizando navegação entre páginas, serviços e persistência de dados.

---

## 🚀 Tecnologias utilizadas

* Angular
* Angular Material
* TypeScript
* RxJS
* Angular Router
* HttpClient

---

## 📌 Funcionalidades

* Cadastro de clientes
* Listagem em tabela (Angular Material Table)
* Edição de clientes via parâmetros de URL
* Exclusão de registros
* Busca de clientes por nome
* Validação e preenchimento de formulário
* Navegação entre páginas (Router)
* Consumo de API de estados (UF) e municípios
* Máscaras em campos de input
* Feedback para usuário (ações de sucesso/erro)

---

## 📁 Estrutura abordada no projeto

* Criação do projeto Angular
* Integração com Angular Material
* Criação de toolbar e navegação
* Forms e binding de dados
* Services para regra de negócio
* Repository de clientes (persistência local)
* CRUD completo (Create, Read, Update, Delete)
* Uso de ActivatedRoute e queryParams
* Uso de HttpClient para APIs externas

---

## ⚙️ Como rodar o projeto

### Pré-requisitos

* Node.js instalado
* Angular CLI instalado

### Instalação

```bash
npm install
```

### Rodar o projeto

```bash
ng serve
```

Acesse no navegador:

```
http://localhost:4200
```
